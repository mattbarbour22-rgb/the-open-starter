# The Open Pick 3 Live

A themed Pick 3 golf pool leaderboard for The Open Championship.

Built from the stable U.S. Open version with:
- admin-only round snapshots
- no public Supabase writes
- previous-round movement arrows
- elimination reasons
- NZ tee-time conversion
- no fake pre-round leaderboard placeholders

The 154th Open Championship is scheduled for Royal Birkdale Golf Club, Southport, Merseyside, England, from 16-19 July 2026.
